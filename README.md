# mattbierner.com

Source for [mattbierner.com](https://mattbierner.com)

## Running locally

This is a Jekyll site published via GitHub Pages.

1. Install dependencies:

	```sh
	bundle install
	```

2. Serve the site:

	```sh
	bundle exec jekyll serve
	```

3. Open <http://127.0.0.1:4000> in a browser.

Pass `--host 0.0.0.0` to make the dev server reachable from other devices on your network.
